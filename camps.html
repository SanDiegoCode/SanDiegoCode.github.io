Summer Camps

This summer, learn how to code

Introduction to Programming Fundamentals

In this camp, students will learn the basics of programming using the language Java. Students will learn the fundamentals of programming, learn how to create a simple text adventure, and will finish the camp with a GUI-based game. They will develop better problem solving skills and the ability to understand any other programming language.

Requirements

  • Students must bring a Mac or Windows laptop to attend
  • Students should be in 6th grade or higher (incoming)

Introduction to Web Dev

Students will be introduced to basic programming concepts as they learn to build their own website, from scratch using HTML5, CSS3, and Javascript during this week long course. They will develop better problem solving skills and the ability to understand any other programming language.

Requirements

  • Students must bring a laptop to attend (Mac, Windows, or Chrome)
  • Students should be in 6th grade or higher (incoming)

Introduction to Python with Simple Game Dev

Learn the principles of Python programming and its syntax. Students who have gotten the feel for coding basics and want to delve deeper into the world of programming will be able to develop simple games with graphics in this camp.

Requirements

  • Students must bring a laptop to attend (Mac, Windows, or Chrome)
  • Students should be in 6th grade or higher (incoming)
